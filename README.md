# nbs-asistan
Asistan is a discord bot in turkish which can help you to improve your guild and have some fun.
NBS Asistan kendi sunucum için geliştirmiş olduğum bize sunucu otomasyonları ve birazcık eğlenmemizde yardımcı olan bir bot

# Installation / Kurulum #

  All you need is Node.js then just use "node bot.js" and bot will start.

  Bilgisayarınıza node.jsyi kurduktan sonra "node bot.js" komutu ile botu başlatabilirsiniz.
